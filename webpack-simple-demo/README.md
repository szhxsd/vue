# demo

> A Vue.js project

## Build Setup

``` bash
$ npm install -g cnpm --registry=https://registry.npm.taobao.org
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build
```

``` bash
├─dist			# 发布目录
│   ├─.gitkeep       
├─src			# 开发目录
│   ├─App.vue		# App.vue组件
│   ├─main.js		# 预编译入口
├─.gitignore
├─.babelrc		# babel配置文件
├─index.html		# 主页
├─package.json		# 项目配置文件
├─README.md  
├─webpack.config.js	 # webpack配置文件

```
